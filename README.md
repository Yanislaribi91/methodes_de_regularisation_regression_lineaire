# methodes_de_regularisation_regression_lineaire
Lors de ce projet nous avons étudiés différentes méthodes de régularisation ( ou pénalisation ) sur une classe de modèles de régression linéaire multiple, après avoir étudié et énoncé les différentes garanties théoriques, nous avons  illustré notre propos sur un jeu de données afin de modéliser l'espérance de vie (moyenne) à la naissance.
